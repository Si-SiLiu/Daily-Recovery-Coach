# Daily Recovery Coach 0.18.0

This is a local **pre-1.0** release snapshot. Daily Recovery Coach is **not a medical device**.

- Version: 0.18.0
- Kubios Data Model: 1.0.0
- Kubios Screenshot Import: 1.2.0
- Recovery Engine: 1.0.0
- Baseline Engine: 1.0.0
- Confidence Engine: 1.0.0
- Local Coach Engine: 1.0.0
- Database Schema: 0.7.0
- Dashboard: 0.9.0
- Cloud AI Runtime: disabled
- AI Model: unreleased

## Summary

Adds a three-layer Kubios HRV model (`raw`, `normalized`, `derived`), explicit
source selection, confirmation-only grouping for complementary screenshots,
core Dashboard cards, a dedicated advanced metrics page, and allowlisted AI
Context/report projections.

## Verification

Migration 0.7.0 created an automatic backup and preserved all recovery,
confidence, and local-coach records byte-for-byte at row serialization level.
The local database passed SQLite integrity checking. Forty-six dedicated data
model tests cover more than 35 requested validation categories. Two approved
local screenshot fixtures achieved 100% template detection and 100% exact
match across 17 visible numeric fields; neither fixture contains a complete
ISO date, so no formal health record was created.

## Privacy and Safety

All processing remains local. Missing fields remain `NULL`; raw OCR text,
screenshots, paths, secrets, and unrestricted payloads are excluded from AI
Context. Advanced metrics require two explicit export confirmations. Recovery,
confidence, and Local Coach formulas and versions are unchanged.
