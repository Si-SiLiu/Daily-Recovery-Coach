## Summary

Calibrates Readiness Summary and HRV Parameters against two user-approved,
cropped, anonymized genuine Kubios screenshots. Valid portrait crops now pass
quality checks, observed anchors drive automatic template selection, and fixed
regions cover only fields actually visible in each layout.

## Verification

The two genuine samples pass template detection and all visible supported
numeric-field comparisons. They do not contain every mandatory import field,
so the required-field import metric remains incomplete and the App correctly
requires manual completion. Results Summary still lacks a genuine calibration
sample. No screenshot was imported into the health database.

## Privacy and Safety

Fixtures contain no identity, notification, or unrelated screen content and
remain local. Evaluation reports aggregate metrics only. No value is guessed,
no complementary screenshots are merged automatically, and final import
confirmation and CSV priority are unchanged.
