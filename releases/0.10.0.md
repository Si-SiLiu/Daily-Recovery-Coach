# Daily Recovery Coach Release

- Version: 0.10.0
- Release Date: 2026-07-10
- Status: Pre-1.0 local release snapshot

## Summary

This release finalizes project governance and release readiness around the
existing local recovery pipeline. It does not change recovery scoring, baseline
calculation, OAuth, data collection, or import behavior.

## Major Features

- Automated machine-to-Markdown current-state synchronization.
- Unified runtime version authority.
- Numeric unittest result semantics with explicit success state.
- Read-only Dashboard System Status and health classification.
- Quality Gate and versioned release-record mechanisms.

## Architecture

The product data path is unchanged. `src/system_status.py` is a read-only
governance sidecar that consumes project state, versions, and the existing
Dashboard database query layer.

## Limitations

- It does not provide One-Click Sync or an AI Coach.
- Validation is currently centered on local operation and limited history.

## Safety Boundary

system. Outputs are informational training-decision support only.
