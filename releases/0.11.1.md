# Daily Recovery Coach Release

- Version: 0.11.1
- Release Date: 2026-07-10
- Status: Pre-1.0 local patch release snapshot

## Summary

This patch closes the Pipeline Architecture Review findings without changing
OAuth, PolarClient requests, imports, recovery formulas, baselines, or the
recovery database schema.

## Major Features

- Required endpoint failures stop and become resumable.
- Optional capability failures become visible warnings.
- Safe pipeline error codes replace opaque exception-only output.
- Final history write failures receive controlled handling.

## Architecture

The fetch adapter consumes structured save results while PolarClient owns all
request behavior. Warning counts flow through the independent operational
history into Dashboard System Status.
