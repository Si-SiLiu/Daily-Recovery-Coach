# Daily Recovery Coach Release

- Version: 0.13.0
- Release Date: 2026-07-10
- Status: Pre-1.0 local release snapshot

## Summary

Implements independent deterministic Recovery Confidence without changing
Recovery Score, recommendations, Baseline Engine, OAuth, or Polar requests.

## Major Features

- Six signal-group completeness model.
- Baseline maturity and 55/45 confidence formula.
- Four confidence levels and explainable group persistence.
- One-Click Pipeline and Dashboard integration.

## Architecture

Confidence is a sidecar reading daily metrics and baselines and writing only its
own versioned table.
