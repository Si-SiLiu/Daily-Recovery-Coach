This release adds default Simple structured-training entry, conditional fields,
RPE/RIR/none preference, catalog auto-fill, explicit custom-library confirmation,
and compact safe actions. Advanced values remain preserved. No schema,
deterministic health algorithm, Polar, or cloud-AI runtime change is included.

Daily Recovery Coach is a general wellness and training decision-support tool,
not a medical device.
