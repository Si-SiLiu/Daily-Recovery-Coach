## Summary

Adds a local, extensible internationalization architecture with Simplified
Chinese and English Dashboard, Daily Log, Local Coach presentation, reports,
and AI Context Markdown. Language preference is stored locally, report output
is separated by locale, and machine-facing JSON/CSV keys remain stable.

No Recovery, Baseline, Confidence, Local Coach, Polar OAuth/API, health
database schema, cloud runtime, or AI model behavior changed. Daily Recovery
Coach remains a pre-1.0 application and is not a medical device.
