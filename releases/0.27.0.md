# Daily Recovery Coach 0.27.0

- Version: 0.27.0
- Database Schema: 0.15.0
- Dashboard: 1.8.0
- Recovery Engine: 1.0.0
- Baseline Engine: 1.0.0
- Confidence Engine: 1.0.0
- Status: local pre-1.0 release

Brand-based supplement logging replaces repeated daily ingredient entry with
versioned product profiles. Schema 0.15.0 adds product, ingredient, intake,
favorite, source-history and candidate tables while retaining legacy fields.

Confirmed products can deterministically calculate active/nutrient intake; an
unconfirmed product remains recordable without fabricated ingredients. Cloud
enrichment remains provider-blocked, and medication is kept separate.

Recovery, Baseline, Confidence and Polar versions/formulas are unchanged.

Daily Recovery Coach is not a medical device.
