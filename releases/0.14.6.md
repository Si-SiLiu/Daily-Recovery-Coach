## Summary

Adds opt-in `--if-new-data` synchronization. The mode hashes only tracked local
Polar source files before and after fetch. When content is byte-identical, no
new raw rows exist, and no Kubios CSV is pending, deterministic rebuild and
report steps are recorded as successful no-data skips while Governance still
updates state.

First runs, any source-content change, new raw rows, pending Kubios input,
dry-run, and ordinary sync all retain the existing full-processing behavior.
No hash or source payload is logged.

Daily Recovery Coach remains a pre-1.0 application and is not a medical device.
