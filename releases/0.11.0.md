# Daily Recovery Coach Release

- Version: 0.11.0
- Release Date: 2026-07-10
- Status: Pre-1.0 local release snapshot

## Summary

Phase 11 adds a single-command, selective, and resumable orchestration layer over
the existing local recovery workflow. Existing engine, OAuth, API, import, and
database business contracts remain unchanged.

## Major Features

- One-Click token-to-governance pipeline.
- Dry Run without database or governance writes.
- Selective Sync for individual steps.
- Resume from the most recent interrupted run.
- Safe lifecycle logging and independent SQLite sync history.
- Read-only Last Sync facts in Dashboard System Status.

## Architecture

`src/sync_pipeline.py` owns ordering only. Single-purpose modules under
`src/pipeline/` invoke existing product-layer functions. Operational history is
stored separately from `recovery.db`.

system. It provides informational training-decision support only.
