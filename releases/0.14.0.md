- Version: 0.14.0

- Dashboard: 0.5.0

## Summary

Adds Local Deterministic Coach v1.0 as a read-only interpretation sidecar after
Recovery and Confidence. It generates structured training, sleep, recovery,
hydration, and directional nutrition advice entirely on-device. No cloud model,
provider adapter, API key, or health-data transfer is introduced.

## Migration

Migration ledger entry `0.4.0` creates `local_coach_recommendations` with
idempotent `(date, engine_version)` storage. The real database was backed up
before migration and protected Recovery, Confidence, and Baseline records were
verified unchanged.

## Product status

Daily Recovery Coach remains a pre-1.0 application and is not a medical device.
Local Coach rules are transparent engineering rules and are not clinically
validated. Cloud AI remains blocked and its model version remains `unreleased`.
