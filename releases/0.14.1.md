## Summary

Adds a read-only Local Coach Longitudinal Evaluation CLI and centralized
evaluation thresholds. It checks stored-output Schema validity, deterministic
regeneration consistency, required safety notices, no-cloud markers, unique
keys, coverage, classifications, and transitions without changing advice.

The real 26-record historical dataset passed every retrospective consistency
check. All records are historical, so prospective validation with fresh daily
records remains the recommended next step.

Daily Recovery Coach remains a pre-1.0 application and is not a medical device.
No scoring formula, database schema, Dashboard analysis, cloud runtime, or Local
Coach rule changed in this patch.
