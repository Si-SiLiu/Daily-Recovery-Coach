## Summary

This release separates long-term sleep regularity from last-night schedule
deviation. It adds a canonical sleep-record boundary, robust circular summary
scoring, timeline-based SRI selection, explicit data maturity and unavailable
states, and regression coverage. The existing UI structure and visual layout
are unchanged. This project is pre-1.0 and is not a medical device.
