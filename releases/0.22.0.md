# Daily Recovery Coach 0.22.0

- Version: 0.22.0
- Maturity: local pre-1.0 release

## Summary

This release adds Personal Information as a top-level App section. It displays
name, gender, date of birth, automatically calculated age, and height; shows the
latest weight, body-fat percentage, and waist measurement; and charts weight
change across the latest 28-day window.

Three separate local forms provide Save Personal Information, Save Body Data,
and Save Targets actions. Targets include optional weight, body-fat percentage,
and waist values. Profile and target data remain on the device.

Basic-information and body-status labels and values are left-aligned within each
column for consistent scanning. The gender form label and selected dropdown value
are centered while the dropdown arrow remains fixed on the right.

## Data Safety

Migration 0.10.0 adds one-row local profile and target tables. Age is calculated
at display time rather than stored. Existing body measurements and source data
are not rewritten, and the migration is preceded by the standard SQLite backup.

## Verification

Validation covers age boundaries, profile and target upserts, value ranges,
latest-body selection, localized navigation, schema history, and regression
safeguards. This personal tracking app is not a medical device.
