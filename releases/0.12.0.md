# Daily Recovery Coach Release

- Version: 0.12.0
- Release Date: 2026-07-10
- Status: Pre-1.0 local release snapshot

## Summary

This release persists and verifies recovery database schema migration history.
It does not change health business columns, recovery formulas, baseline behavior,
OAuth, Polar requests, imports, or reports.

## Major Features

- Ordered schema migration ledger inside `recovery.db`.
- SemVer, stable names, immutable checksums, and applied timestamps.
- Automatic baseline registration for legacy databases.
- Strict config-to-ledger version verification.
- Genuine read-only Dashboard SQLite connections.

## Architecture

Write-side `db.connect/init_db` owns migrations. Read-only consumers use SQLite
URI `mode=ro`. Project governance measures the ledger through read-only queries.
