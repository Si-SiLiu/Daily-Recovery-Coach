# Manual ChatGPT Sync

Supported flow: `Local Export → User Preview → Explicit Confirmation → Manual Upload`.

The system does not sign in to ChatGPT, access cookies, automate a browser,
accept an API key, call an AI API, or upload files. The user must inspect every
export. After manual upload, the target service's file and chat data policies
apply. AI analysis cannot replace a doctor, registered dietitian, or qualified coach.
